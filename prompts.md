# prompts.md  
## Prompt Engineering Workflow – Brew Haven Cafe

This document contains the structured prompts used to design and refine the Brew Haven Cafe website.

The prompts follow an iterative refinement model to improve both layout and UI quality step by step.

---

# Prompt 1 – Generate Café Website Layout

## Objective:
Create the base structure and design.

### Prompt Used:

You are a professional front-end web developer and UI designer.

Create a clean, modern café website using HTML and CSS.

Business Name: Brew Haven Cafe  
Location: Hyderabad  
Brand Tone: Warm, premium, modern  
Target Audience: College students, remote workers, young professionals  

The homepage must include:
- Fixed navigation bar
- Hero section with welcoming headline
- About section
- Services section
- Booking section
- Contact section

Design Requirements:
- Premium and elegant layout
- Soft and warm color palette
- Clear typography hierarchy
- Balanced spacing between sections
- Fully responsive design

Ensure:
- Code is clean and structured
- Proper indentation
- No unnecessary complexity

---

# Prompt 2 – Add Background Image Slider

## Objective:
Enhance hero section without breaking layout.

### Prompt Used:

Modify only the hero section of the existing Brew Haven Cafe website.

Add:
- Automatic background image slider
- Smooth fade transition
- Image change every 3–5 seconds

Constraints:
- Do NOT remove any sections
- Do NOT modify navigation structure
- Maintain fixed navbar
- Preserve layout consistency

Ensure:
- Smooth animation
- No layout shift
- Professional premium look
- Mobile compatibility

---

# Prompt 3 – Improve UI Styling & Visual Polish

## Objective:
Upgrade overall visual quality and user experience.

### Prompt Used:

Improve the UI styling of the Brew Haven Cafe website.

Enhance:
- Section spacing and padding
- Button hover animations
- Smooth transitions
- Subtle shadow effects
- Visual consistency across sections

Make the website:
- More premium
- Visually balanced
- Suitable for a modern Hyderabad café

Ensure:
- Soft warm color palette
- Clear heading hierarchy
- Consistent margins
- Fully responsive design

---

# Prompt Engineering Approach

This project followed a structured refinement workflow:

1. Context Definition  
2. Layout Generation  
3. Controlled Feature Addition  
4. UI Enhancement  
5. Final Optimization  

Each iteration improved output clarity, functionality, and visual appeal while maintaining stability.

This demonstrates how structured prompting directly improves development quality.
